📷 Photo Import

Scan a filled paper log sheet and add the activities to your Activity Register · ← Back to dashboard
100% local: your photo is processed entirely in this browser (perspective correction, checkbox detection, and text recognition all run on-device). The only network activity is a one-time download of the open-source text-recognition library from a public CDN — the photo itself is never uploaded anywhere.
Handwriting recognition is imperfect — expect to correct some fields by hand. Checkbox fields (Priority, Type, R&D) are detected by simple mark/no-mark image analysis and are far more reliable than handwritten text. Every row is shown for review before anything is saved, and importing here always adds to your existing activities — it never replaces or deletes them.

1Load Photo

Tap to take a photo or choose an image of the filled sheet

2Align the Table

Drag the 4 orange handles onto the 4 outer corners of the data rows only — start just below the column-header text, and end at the bottom-right of your last filled row. Don't include the title/header row.

3Confirm & Extract

4Review & Add

This tool reads and writes the same local storage as the main Activity Register on this device/browser — nothing is sent to a server. Import here only ever adds new activities; it never deletes or overwrites existing ones.
